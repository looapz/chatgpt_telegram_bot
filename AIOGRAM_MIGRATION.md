# Миграция на Aiogram 3.x

## Обзор изменений

В рамках обновления кодовой базы проекта был осуществлен переход с библиотеки python-telegram-bot на aiogram версии 3.18.0. Это позволяет:

- Использовать современный асинхронный API Telegram
- Повысить производительность и надежность бота
- Использовать более удобные паттерны и структуры для обработки сообщений
- Легче масштабировать функциональность в будущем

## Основные технические изменения

1. **Замена базовых компонентов**:
   - Замена `Application` и `ApplicationBuilder` на `Dispatcher`
   - Использование системы роутинга aiogram 3.x
   - Переход на модель фильтров aiogram

2. **Обновления синтаксиса**:
   - Использование новой структуры обработчиков
   - Использование `InlineKeyboardBuilder` для создания клавиатур
   - Использование `Enum` для констант ParseMode

3. **Изменения в API**:
   - Изменение работы с файлами (FSInputFile)
   - Новый подход к обработке колбеков
   - Современные типы данных для сообщений и взаимодействий

## Примеры изменений кода

### Старый код (python-telegram-bot):
```python
application = (
    ApplicationBuilder()
    .token(config.telegram_token)
    .concurrent_updates(True)
    .rate_limiter(AIORateLimiter(max_retries=5))
    .http_version("1.1")
    .get_updates_http_version("1.1")
    .post_init(post_init)
    .build()
)

application.add_handler(CommandHandler("start", start_handle, filters=user_filter))
```

### Новый код (aiogram 3.x):
```python
bot = Bot(token=config.telegram_token)
dp = Dispatcher()

dp.message.register(start_handle, CommandStart())
```

## Преимущества миграции

- **Повышение производительности**: aiogram 3.x оптимизирован для работы с большим количеством запросов.
- **Удобство разработки**: более чистый и понятный код, лучшая поддержка типов.
- **Поддержка современных функций Telegram API**: полная поддержка последних возможностей Telegram.
- **Лучшая масштабируемость**: более гибкая архитектура для расширения функциональности бота.
- **Активное сообщество**: aiogram имеет активное русскоязычное сообщество разработчиков.
